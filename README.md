## :blue_book: **Deskripsi**
<p align="justify">
Eventbrite adalah platform teknologi acara terbesar di dunia, yang memotori jutaan acara di lebih dari 180 negara dan wilayah. Halaman ini menyajikan informasi terkait Event - event nasional berdasarkan nama event, tempat dan waktu pelaksanaan. 
</p>

<p align="justify">
Project ini merupakan scraping data pada website https://tiktokhashtags.com/ ,dimana yang discraping adalah informasi terkait event - event nasional yang akan dilaksanakan di dalam negeri, sehingga memudahkan seseorang dalam mencari informasi terkait kegiatan atau event yang akan dilaksanakan.
</p>
